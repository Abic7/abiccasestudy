What the v4 rebuild replaced
23 nodes: 4× Filter, 4× Sort, 3× Switch, 4× Prepare, 4× LM Studio, 4× Parse, 1× Merge. Replaced with 1 dynamic Prepare node + a lookup table. Any new call type is handled with a config entry — no new nodes required.
Parse node reliability fixes
Every LLM node has a matching Parse node that: strips Gemma <think> tags, repairs truncated JSON (unclosed brackets), removes phantom trailing ], and falls back to reasoning_content if content is empty. Zero parse failures across all 18 transcripts.
